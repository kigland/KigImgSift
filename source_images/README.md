# Source Images Directory

This directory should contain the images you want to classify.

## Supported formats:
- JPG/JPEG
- PNG
- WebP

## How to add images:
1. Copy your image files to this directory
2. Start the backend server
3. Launch the frontend application
4. Use keyboard shortcuts to classify images:
   - Press '1' for frontal faces
   - Press '2' for side faces
   - Press 'Space' to skip

## Example:
Place files like:
- photo1.jpg
- portrait.png
- image001.webp

The application will automatically scan this directory and display images for classification.
